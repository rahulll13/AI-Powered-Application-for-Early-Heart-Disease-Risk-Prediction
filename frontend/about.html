About the Project

This application is a full-stack project designed to predict the likelihood of heart disease based on patient data. It leverages a machine learning model served via a Python backend and an interactive frontend for user management and data input.

Backend Technology

  • Framework: Flask
  • API: Flask-RESTful
  • Database: SQLite with Flask-SQLAlchemy
  • Authentication: JWT with Flask-JWT-Extended
  • Machine Learning: Scikit-learn
  • OCR: Tesseract (via pytesseract)

Frontend Technology

  • Structure: HTML5
  • Styling: Bootstrap 5 & Custom CSS
  • Logic: Vanilla JavaScript (ES6+)
  • API Communication: Fetch API
